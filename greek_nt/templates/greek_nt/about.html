About Greek NT Search

Greek NT Search is a tool for searching and studying the Greek New Testament. It provides instant search across Greek text, English glosses, and Strong's numbers, with interlinear viewing options.

Text and Data Sources

SBL Greek New Testament

This site uses the SBL Greek New Testament, a critically edited text by Michael W. Holmes, produced through collaboration between the Society of Biblical Literature and Logos Bible Software. The SBLGNT represents a significant scholarly achievement, differing from the Nestle-Aland/United Bible Societies text in more than 540 variation units.

MACULA Greek Linguistic Datasets

The linguistic data (morphology, glosses, etc.) comes from the MACULA Greek Linguistic Datasets. This comprehensive dataset combines the SBLGNT with:

  • Morphological analysis
  • English glosses from the Berean Study Bible
  • Word sense data from the UBS MARBLE project
  • Semantic roles and participant referents
  • Syntax trees and linguistic annotations

SDBG Greek Definitional Data

The contextual definitions come from the Semantic Dictionary of Biblical Greek (SDBG). This innovative resource provides:

  • Context-specific definitions for each word
  • Precise semantic domain classifications
  • Multiple specialized glosses per word
  • Clear, modern definitional language

Technical Details

This is an open-source project built with Django and Tailwind CSS. The search functionality provides:

  • Full-text search across Greek text, lemmas, and English glosses
  • Strong's number lookup
  • Interactive contextual definitions for each Greek word
  • Interlinear display with morphological information
  • Fast, cached results with pagination
  • Semantic domain classifications for each word
